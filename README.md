# Stanley Xie's Homebrew Tap

This tap contains Homebrew formulae for StanleyXie's projects.

## Installation

```bash
brew tap stanleyxie/tap
```

## Available Formulae

### tfpipboy

Context-aware Terraform CLI tool with real-time status display.

```bash
brew install stanleyxie/tap/tfpipboy
```

## Usage

After installation, you can run:

```bash
tfpipboy --help
```

## Updating

```bash
brew update
brew upgrade tfpipboy
```

## Repository

This tap is automatically updated by GoReleaser when new releases are published.
